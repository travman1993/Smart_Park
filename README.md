# SmartPark

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Status](https://img.shields.io/badge/Status-Active%20Development-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

SmartPark is a modular smart parking infrastructure simulation platform built in Python. The project focuses on occupancy tracking, AI-assisted vehicle workflows, gate logic systems, parking analytics and future hardware integrations.

The system is designed to simulate real parking technologies used in:

- Residential communities
- Business parking lots
- Event parking system
- Parking garages and parking decks
